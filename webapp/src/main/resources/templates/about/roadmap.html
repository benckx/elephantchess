Roadmap

Here are feature ideas we are currently considering. Use thumbs up/down to vote on each one and click Tell us more after voting if you want to share more details.

Analysis for Manchu variant

Extend analysis board support to the Manchu variant (that would be with Fairy Stockfish).

Mini Xiangqi variant

Add Mini Xiangqi variant.

Janggi

Add Janggi (Korean chess).

Other games or variant?

Please tell us more.

Game of the Seven Kingdoms

The Game of the Seven Kingdoms is a xiangqi variant that include 7 players. The plan would be to also allow players to play multiple armies, so they wouldn't have to find 7 people to play but just 2, 3 or 4. Later we would also consider adding a 8 armies variant, so the game would be more balanced with 2, 4 or 8 players.

Some work on this variant project has already been done but it's not yet possible to play live online.



New Types of Puzzles

More varied type of puzzles than "find the shorted path to mate", more like "find the best move" in this situation.

Better Puzzle Experience on Mobile

Make the puzzle page more intuitive on mobile, with a dedicated action block at the bottom of the screen (next puzzle, watch the solution, re-attempt).



Pre-move in games

Add a pre-move system for live games so players can queue one move while waiting for the opponent's move.

Basic Cheating Detection

Add a basic cheating detection that checks, for each player-vs-player game, whether the moves match a player-vs-bot game.



Emoji Reactions in Chat

Add an emoji box to the in-game chat so players can quickly react during their games.

Profanity filter in chat

Have an option to see or not see offensive words. In English or maybe in other languages.

Anti-spamming in Chat

Throttle spammers in the chat, avoid receiving the same annoying message in a short span of time. Or maybe even mute the chat.

Chat box in the Lobby

Add a little chat box in the Lobby (i.e. on the home page).



Time Control in PvB

Play-vs-Bot has no time limit right now (beside the auto-cancel after 2 or 7 days of inactivity). We could add time control to PvB.

Retakes in PvB

Allow player to backtrack one or multiple steps when playing against the bot.



Vertical Evaluation Bar

Vertical orientation for the engine evaluation bar on the analysis board, or when the analysis data is visible on PvP, PvB or DB pages.



Opening by Player

Show most played opening on Database player page. That would also allow to play in PvB against a certain player opening later.

Opening Classification

Align the opening repertoire data (visible in the Analysis Board, on the top left) with established opening classification system like ECCO.

Search in Events and Players Lists

Add a search box to the events list and the players list to make it easier to find a specific event or player.

Media on Event Pages

Allow embedding videos and other media (for example YouTube broadcasts) on event pages.



Articles and Blogs posts

Allow users to create their articles or their own little blog on elephantchess.io.

Tournaments

Add the possibility to create and host tournaments.
