Roadmap

Here are feature ideas we are currently considering. Use thumbs up/down to vote on each one and click Tell me why after voting if you want to share more details.

Pre-move support in games

Add a pre-move system for live games so players can queue one move while waiting for the opponent.

Analysis for Manchu variant

Extend analysis board support to the Manchu variant.

Mini Xiangqi variant

Add support for a mini Xiangqi variant with adapted board rules and game flow.

Implement the Game of the Seven Kingdoms for live PvP?

The Game of the Seven Kingdoms is a xiangqi variant that include 7 players. The plan would be to also allow players to play multiple armies, so they wouldn't have to find 7 people to play.

Some work on this variant project has already been done but it's not yet possible to play live online.
