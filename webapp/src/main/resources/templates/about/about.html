About

The initial motivation behind this webapp was to create a nice, dynamic GUI to play xiangqi puzzles, which didn't seem available at the time. We currently have about 18,000 puzzles available. As of now, all the puzzles are "find the shortest path to mate" (which is the most straight forward to infer from existing games data), but we're planning to create puzzles in other categories as well.

On top of that - with the idea of bringing xiangqi to a wider audience - we added the following:
  • Player-vs-Player (PvP)
  • Player-vs-Bot (PvB)
  • Analysis Board
  • Game Database

We're planning to keep the app ad free and free to use - and possibly open source. In the future we hope user contributions will be sufficient to cover server costs.
