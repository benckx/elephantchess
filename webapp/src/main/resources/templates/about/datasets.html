Datasets

Games played by users of the webapp are public and data is available under GPL-3.0 license. The following datasets are anonymized. Non-anonymized game data can be shared with third parties for research or technical purposes (non-anonymized meaning with usernames, but without email address, IP or any personal data).

The ZIP file contains CSV with 1000 games each. When the game was analyzed, engine data is also available in the CSV.
