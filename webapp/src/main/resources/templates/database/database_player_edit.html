Canonical Name

  • Canonical Name means a "clean" and standardized reference name that can be re-used across the Database. There are used for URL permalink, so avoid changing it too often. For that reason, we'll also avoid accentuated transliterations (e.g. "Fù Guāngmíng").
  • In Chinese names, the surname (i.e. family name) comes first, followed by the given name. Standard Hanyu Pinyin rules dictate that the given name should be written as a single word with only the first letter capitalized. For example: "Wang Tianyi" instead of "Wang TianYi".
  • For Western names, please follow the standard English naming conventions, with the given name first and the surname last. For example: "John Smith".
